# Using Cursor

Cursor is your code editor and AI guide in one. It's where you'll write code, read lessons, and talk to the AI that helps you through this course.

This guide covers the basics you need to get started.

---

## The Interface

When you open Cursor, you'll see:

- **Sidebar (left):** Your file explorer. This is where you navigate folders and files.
- **Editor (center):** Where you read and edit files. You can have multiple files open in tabs.
- **Chat Panel (right or bottom):** Where you talk to the AI.

If you don't see the chat panel, press `Cmd + L` (Mac) or `Ctrl + L` (Windows/Linux) to open it.

---

## How to Talk to the AI

The chat panel is where you'll spend a lot of time. You can:

- Ask questions about what you're learning
- Get help when you're stuck
- Ask the AI to explain code
- Request guidance on what to do next

Just type naturally. The AI is designed to guide you, not rush you.

### Good examples:
- "I just finished the project brief."
- "I'm confused about what `<head>` does."
- "Can you explain this in simpler terms?"
- "What should I do next?"

### Pro tip:
Be specific about what file you're working on and what's confusing. The more context you give, the better the help.

---

## Referencing Files with @

This is important: **you can reference files directly in your messages using the `@` symbol.**

For example:
- `@index.html` — references a file called index.html
- `@02_LEARNER_STATE/goals.md` — references a specific file in a folder
- `@04_PROJECTS/` — references an entire folder

When you use `@`, Cursor shows you a dropdown of matching files. Select the one you want, and the AI will be able to see its contents.

This is useful when:
- You want to ask about a specific file
- You want the AI to check your work
- You're referencing something from the curriculum

## Chat Modes

Cursor has different chat modes. You'll mainly use two:

### Ask Mode
- **What it's for:** Questions, explanations, exploring code
- **What it does:** The AI can read files and answer questions, but it won't make changes
- **When to use it:** When you want to understand something without changing anything

### Agent Mode (also called Normal mode)
- **What it's for:** Getting things done — writing code, creating files, making edits
- **What it does:** The AI can read files *and* make changes with your approval
- **When to use it:** When you're ready to build or edit something

You can switch modes using the dropdown at the top of the chat panel.

**Rule of thumb:** 
- Learning or confused? Use **Ask**.
- Ready to build? Use **Agent**.

## Opening Files

To open a file:
1. Find it in the sidebar (left panel)
2. Click it once to preview, or double-click to keep it open

You can also use `Cmd + P` (Mac) or `Ctrl + P` (Windows/Linux) to quickly search for and open files by name.

---

## Viewing Your Website

When you create HTML files, you'll want to see them in a browser.

**Option 1: Drag and drop**
- Find your `.html` file in Finder (Mac) or File Explorer (Windows)
- Drag it into your browser window

**Option 2: Right-click in Cursor**
- Right-click the file in the sidebar
- Select "Reveal in Finder" (Mac) or "Reveal in File Explorer" (Windows)
- Double-click the file to open it in your default browser

**Option 3: Live Server (later)**
- Once you're more comfortable, you can install the Live Server extension for auto-refreshing previews
- We'll cover this later — don't worry about it now

## If the Chat Gets Confusing

Sometimes a chat thread gets long or goes off track. You can:

- **Start a new chat:** Click the `+` icon in the chat panel
- **Keep going:** The AI reads your learner state files, so it can pick up context even in a new chat

Your progress is stored in files, not in the chat. A new chat doesn't mean lost progress.

## Keyboard Shortcuts Worth Knowing

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Open chat panel | `Cmd + L` | `Ctrl + L` |
| Quick file search | `Cmd + P` | `Ctrl + P` |
| Save file | `Cmd + S` | `Ctrl + S` |
| Toggle sidebar | `Cmd + B` | `Ctrl + B` |

You don't need to memorize these. Just know they exist.

## Summary

- **Sidebar** = files. **Editor** = reading/writing. **Chat** = AI help.
- Use `@` to reference files in your messages.
- **Ask mode** for learning. **Agent mode** for building.
- Your progress lives in files, not chat history.
