Use this prompt to generate a clean, minimal 4-page Jekyll site with a blog, about page, and contact form. Perfect for getting a project off the ground in minutes.

```markdown
Act as an expert Jekyll developer. Create a file structure for a new Jekyll site with the following pages:
1. Home (index.md)
2. About (about.md)
3. Blog (blog/index.html)
4. Contact (contact.md)

Use a shared default layout. Include a navigation include and a footer include.
Provide the CSS for a minimal, dark-themed aesthetic.
```
